# Environment Setup

Before getting started with Linux, we need to prepare the development environment. This tutorial uses the following software throughout all subsequent chapters.

## VMware Workstation

VMware Workstation is a professional virtualization platform that allows you to create and run virtual machines on your local computer. It provides an isolated and secure environment for installing and experimenting with Linux without affecting your host operating system.

Purpose

- Create and manage Linux virtual machines
- Save and restore snapshots
- Provide a stable environment for hands-on practice

---

## CentOS 7

CentOS 7 is a classic enterprise-grade Linux distribution known for its stability, mature ecosystem, and extensive documentation. It is an excellent choice for beginners learning Linux fundamentals and basic system administration.

Purpose

- Learn Linux fundamentals
- Practice common Linux commands
- Build a server-like environment for experiments

---

## Xshell

Xshell is a powerful SSH terminal emulator for Windows. It offers a convenient way to remotely connect to and manage Linux systems, making command-line operations more efficient than working directly inside the virtual machine.

Purpose

- Connect to Linux via SSH
- Execute Linux commands remotely
- Improve workflow and productivity
