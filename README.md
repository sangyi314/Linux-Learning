# Linux Learning Log — Starting from Absolute Zero

This repository is a **full learning log** from someone with **no prior Linux experience**, documenting the journey from the very beginning.

I had almost no hands-on Linux background before this. Starting from installing a system, opening a terminal, and typing the first command, I try to record honestly what I learn each day (or each session): concepts, mistakes, commands I practice, and small scripts—synced to GitHub for my own review and for anyone else starting from zero.

## About This Repository

- **Purpose**: Personal notes, practice code, and post-mortems—not an official course or a complete curriculum.
- **Honest record**: Includes things I didn’t understand, wrong commands, and how I looked things up—not only polished “right answers.”
- **Ongoing**: Learning doesn’t end; this repo will grow over time.
